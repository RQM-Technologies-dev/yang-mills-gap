# Problem Statement

The Clay Yang-Mills existence and mass gap problem asks for a rigorous construction of quantum Yang-Mills theory on four-dimensional spacetime and a positive mass gap.

This repository is not that result. It creates a finite-lattice Python sandbox for exploring SU(2) Wilson lattice Yang-Mills in unit-quaternion coordinates.

The exploratory RQM thesis is:

> "The Yang-Mills mass gap may be interpreted in Resonant Quantum Mechanics as the lowest nonzero energy of a closed, gauge-invariant non-abelian curvature resonance."

The first research task is standard:

- represent SU(2) links as unit quaternions,
- compute plaquette holonomy,
- compute Wilson action,
- measure gauge-invariant observables,
- form glueball-like connected correlators.

RQM language is applied as interpretation. It is not inserted into the baseline action.
