# Closure Defect And Wilson Action

The plaquette is:

```text
U_mu(x) U_nu(x + mu) U_mu^-1(x + nu) U_nu^-1(x)
```

The closure defect is:

```text
D_p = 1 - scalar_part(U_p)
```

The Wilson action is:

```text
S = beta * sum_p D_p
```

The RQM interpretation reads plaquette holonomy as curvature closure and the Wilson action as closure cost. This interpretation does not alter the standard Wilson action.
